- Dockerfile is found at eiffel-vici\app\src\main\docker\Dockerfile
- Docker image is created with a gradle docker plugin, found in eiffel-vici\app\build.gradle

1: Unzip eiffel-vici\app\src\main\resources\static\events.zip into eiffel-vici\app\src\main\resources\static\events.json

2: Move to directory eiffel-vici\app

3: Test if build and run works correctly:
	Windows PowerShell: 	(./gradlew build) -and (java -jar .\build\libs\vici-eiffel-0.1.jar)
	Linux: 					./gradlew build && java -jar .\build\libs\vici-eiffel-0.1.jar

4: Build docker image:
	./gradlew build buildDocker

5: Run docker image on http://localhost:8080/:
	docker run -p 8080:8080 -t ewahjon/vici-eiffel:0.1


Stop running container:
1:	docker ps
2:	docker kill [CONTAINER ID]
